Quick start

https://selectx.github.io/1.0.0/selectx.css

https://selectx.github.io/1.0.0/selectx.js

    $('.selectx').selectx();

Settings

    {
        placeholder: 'Choose value',
        noResults: 'Not found',
        cssClass: 'selectx-default',
        search: true
    }

Knowing bugs that will not be ever fixed

  • Backslashes in search "\" are ignored
  • If mouse over results list, arrow navigation may jump a little

How to

  • If deprecated DOM mutation events does not work, to update SelectX after native select changes, use
    $('#my-select').data('selectx').init()
